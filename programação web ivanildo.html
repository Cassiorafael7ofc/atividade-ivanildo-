✅ Resumo da Proposta: Landing Page – EstudaMax

🎯 Objetivo:

Criar uma landing page moderna, atrativa e responsiva com o Bootstrap, para apresentar um app fictício de organização de estudos por IA chamado "EstudaMax", com foco em converter visitantes em leads.

🔧 Requisitos Técnicos da Página:

1. Barra de Navegação (Navbar)

  • Tipo: Fixa no topo ou colapsável (hamburger menu) para mobile.

  • Elementos:

    • Logo do EstudaMax.

    • Links âncora para seções da página: #sobre, #funcionalidades, #planos, #contato.

💡 Dica: Use <nav class="navbar navbar-expand-lg navbar-light bg-light fixed-top">.

2. Seção Hero (Topo)

  • Destaque principal do app.

  • Deve conter:

    • Imagem ou vídeo de fundo.

    • Título chamativo (ex: "Estude com Inteligência").

    • Breve descrição.

    • Botão de CTA (ex: “Experimente Grátis”).

💡 Dica: Use container, row, col e btn btn-primary ou btn-outline-*.

3. Seções de Conteúdo

A. Funcionalidades/Benefícios

  • Utilize Cards ou Grid para organizar os conteúdos.

  • Inclua ícones ilustrativos para:

    • Planejamento de estudos

    • Notificações inteligentes

    • Análise de desempenho

    • Personalização por perfil

💡 Dica: Bootstrap Icons ou FontAwesome (<i class="bi bi-alarm"></i>).

B. Planos e Preços

  • Use Cards ou Tabela (Table) para comparar os planos:

    • Gratuito

    • Premium

    • Profissional

  • Mostrar:

    • Preço

    • Funcionalidades incluídas

    • Botão de assinatura

💡 Dica: Destaque o plano intermediário com uma borda ou cor diferente.

C. Depoimentos

  • Use o componente Carousel do Bootstrap.

  • Pelo menos 3 depoimentos de usuários fictícios, com:

    • Nome

    • Foto (pode ser placeholder)

    • Comentário

💡 Dica: <div id="carouselTestimonials" class="carousel slide">.

4. Formulário de Contato/Leads

  • Campos: Nome, E-mail, Mensagem.

  • Validação com required, type="email", etc.

  • Layout agradável com form-group, form-control.

💡 Dica: Pode usar form-floating do Bootstrap 5 para aparência moderna.

5. Rodapé (Footer)

  • Deve conter:

    • Links úteis (Termos, Política, etc.)

    • Contatos da startup

    • Direitos autorais

💡 Dica: Use bg-dark text-light e container py-4.

🧩 Sugestão de Estrutura HTML com Bootstrap:

html
<body data-bs-spy="scroll" data-bs-target=".navbar" data-bs-offset="70">
  <!-- Navbar -->
  <!-- Hero Section -->
  <!-- Funcionalidades -->
  <!-- Planos e Preços -->
  <!-- Depoimentos -->
  <!-- Formulário de Contato -->
  <!-- Footer -->
</body>

📝 Dicas Finais:

  • Utilize CDN do Bootstrap para iniciar rápido.

  • Garanta que a página esteja responsiva (verifique em mobile/tablet).

  • Pode usar ferramentas como Bootstrap Studio ou Figma para prototipar.

  • Comente seu código para facilitar a leitura.

  • Teste o formulário com validação nativa do HTML e visual moderno.
